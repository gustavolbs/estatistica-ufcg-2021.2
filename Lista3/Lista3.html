Questão 4

Mostre que, em qualquer distribuição normal, à área sob a curva (probabilidade), determinada por cada um dos intervalos abaixo, é sempre a mesma e independente dos parâmetros da distribuição

a) (µ − 1, 64σ; µ + 1, 64σ);

Resposta

para simplificar atribuimos x = µ − 1, 64σ e y = µ + 1, 64σ

portanto :

Zy = 1,64 e Zx = - 1.64

resolvendo P(Zx < Z < Zy)

P(Zx < Z < Zy) = P(-1,64 < Z < 1,64)

. = P(Z < 1,64) - P(Z < - 1,64)

. = 0.9495 - 0.0505 = 0.8990

b) (µ − 1, 96σ; µ + 1, 96σ);

Resposta

para simplificar atribuimos x = µ − 1, 96σ e y = µ + 1, 96σ

portanto :

Zy = 1, 96 e Zx = - 1, 96

resolvendo P(Zx < Z < Zy)

P(Zx < Z < Zy) = P(-1, 96 < Z < 1, 96)

. = P(Z < 1, 96) - P(Z < - 1, 96)

. = 0.9750 - 0.0250 = 0.9500

Resposta
