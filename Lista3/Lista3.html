Questo 4

Mostre que, em qualquer distribuio normal, rea sob a curva (probabilidade), determinada por cada um dos intervalos abaixo, sempre a mesma e independente dos parmetros da distribuio

a) ( 1, 64; + 1, 64);

Resposta

para simplificar atribuimos x = 1, 64 e y = + 1, 64

portanto :

Zy = 1,64 e Zx = - 1.64

resolvendo P(Zx < Z < Zy)

P(Zx < Z < Zy) = P(-1,64 < Z < 1,64)

. = P(Z < 1,64) - P(Z < - 1,64)

. = 0.9495 - 0.0505 = 0.8990

b) ( 1, 96; + 1, 96);

Resposta

para simplificar atribuimos x = 1, 96 e y = + 1, 96

portanto :

Zy = 1, 96 e Zx = - 1, 96

resolvendo P(Zx < Z < Zy)

P(Zx < Z < Zy) = P(-1, 96 < Z < 1, 96)

. = P(Z < 1, 96) - P(Z < - 1, 96)

. = 0.9750 - 0.0250 = 0.9500
